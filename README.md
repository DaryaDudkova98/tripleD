# tripleD — Landing page веб-разработчика

Лендинг для привлечения клиентов на разработку сайтов. Содержит информацию об услугах, ценах, портфолио и калькулятор стоимости.

## Цель проекта

Привлечение клиентов на разработку сайтов через:
- Презентацию услуг и цен
- Демонстрацию возможностей (портфолио)
- Удобный калькулятор стоимости
- Формы связи и контакты

## Технологии

|          Категория          |         Технологии          |
|-----------------------------|-----------------------------|
| **Backend**  | Laravel Framework 13.5.0, PHP 8.3.28       |
| **Frontend** | HTML5, CSS3, Tailwind CSS, JavaScript ES6+ |
| **Сборка**   | Vite                                       |
| **Анимации** | CSS Animations, Intersection Observer API  |

## Структура лендинга

|     Секция      |      ID     |                   Описание                     |
|-----------------|-------------|------------------------------------------------|
| **Hero**        | `#hero`     | Главный экран с призывом к действию            |
| **Обо мне**     | `#about`    | Информация о разработчике                      |
| **Портфолио**   | `#projects` | Примеры работ (пополняется)                    |
| **Услуги**      | `#service`  | Типы сайтов: визитка, лендинг, многостраничник |
| **Калькулятор** | `#pricing`  | Расчёт стоимости проекта                       |
| **Контакты**    | `#contacts` | Связь через мессенджеры и формы                |
